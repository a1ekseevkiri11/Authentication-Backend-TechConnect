## Настройка и запуск проекта

Если не установлен пакетный менеджер ```poetry```, выполните команду:
```
pip install poetry
```
Необходимо создать и активировать виртуальное окружение:
```
poetry shell
```
Далее установите зависимости:
```
poetry install
```
